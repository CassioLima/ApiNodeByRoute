## Projeto Node (API)
Projeto criado para demonstrar como podemos criar uma API em node Consumindo 2 bancos de bados: SqLite E Mongo.

## Instalação
* npm install
* npm install sqlite --save
* npm install -g nodemon
* npm install --save-dev nodemon
